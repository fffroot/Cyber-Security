## 📖 О репозитории
`Cyber-Security` — единое пространство для изучения, тестирования и разработки инструментов информационной безопасности.  
Каждый подпроект **автономен**: имеет собственные зависимости, конфигурацию и документацию. Объединяет их только фокус — анализ уязвимостей, защита данных, расследование инцидентов и безопасная автоматизация.
